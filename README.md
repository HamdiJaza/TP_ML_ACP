
# TP_ACP_F

This repository contains resources for analyzing the World Happiness Report dataset using Principal Component Analysis (ACP) in a Jupyter notebook.

## Contents
- `TP_ACP_F.ipynb`: Main notebook for data analysis and ACP implementation.
- `World Happiness Report data.csv`: Dataset used in the analysis.

## How to Use
1. Open the folder in VS Code or Jupyter.
2. Run `TP_ACP_F.ipynb` with a Python environment that includes the following packages:
   - pandas
   - numpy
   - scikit-learn
   - matplotlib
   - seaborn

### Install Required Packages
Use the following command in PowerShell:
```powershell
python -m pip install pandas numpy scikit-learn matplotlib seaborn
```
